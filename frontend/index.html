Congresional Predictions

Predicting Congresional Race

This project aims to predict the congresional electional based on 2018 congresional election results and demographic data. We trained the data using a neural network with two layers in order to predict statistically significant demographics on the election and display the results. Additionaly, we ran random forest to generate which fields held the most statistcal significance.
