Congresional Predictions

Predicting 2020 Congresional Race

This project aims to predict 2020 congresional electionsl based on 2018 congresional election results and demographic data. We trained the data using a neural network with two layers in order to predict 2020 elections and display the results. Additionaly, we ran random forest to generate which fields held the most statistcal significance.
